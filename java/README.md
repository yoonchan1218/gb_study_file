# java

JAVA 기초 문법부터 객체지향, 컬렉션, JDBC까지 정리한 폴더이다.
처음에는 언어 개념을 익히는 데 집중했고, 뒤로 갈수록 클래스와 상속,
인터페이스, 컬렉션, DB 연결 순서로 공부했다.

## 정리한 내용

- JAVA 기본 구조와 실행 원리
- 출력, 변수, 형변환, 상수, 입력, 연산자
- 제어문, 배열, 메서드
- 클래스, 상속, 다형성, 추상화, 인터페이스
- Object 클래스와 Collection Framework
- JDBC 기초

## 폴더 구성

- `memo/`: 단원별 메모 정리
- `강의자료/`: 수업 자료와 JDBC 참고 파일
- `resource/`: 설치 파일이나 실행 환경 자료
- `worspace/`: 이클립스 작업 흔적

## 확인한 메모

- `memo/a_intro.txt`
- `memo/b_print.txt`
- `memo/c_variable.txt`
- `memo/h_control_statement.txt`
- `memo/u_collection_framework.txt`

## 공부 방식

단원별로 파일을 나눠서 복습하기 쉽게 정리했다.
정의부터 적고, 필요한 경우 잘못 이해하기 쉬운 부분은 `(x)`, `(o)`처럼
표시해서 다시 확인할 수 있게 적어두었다.
