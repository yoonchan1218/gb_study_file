JAVA - 프로그래밍 언어
	프로그래밍 언어는 개발자와 컴퓨터가 소통하기 위한 언어이다.
                                                    (x)
-->개발자와 운영체제(OS)가 소통(o)

소스코드: 개발자와 컴퓨터가 소통할 것을 글로 작성해 놓은것.
                           (x)
-->개발자와 운영체제(OS)가 소통(o)

소스파일(.java): 소스코드를 작성해 놓은 파일
컴파일: 사람의 언어를 컴퓨터 언어로 바꿔주는 작업(0,1)

		컴파일러는 위에서 아래로, 좌에서 우로 해석한다.

-----------------------------------------------------------------------------------------
JVM(Java Virtual Machine): JAVA 프로그램을 실행해줌.
JRE(Java Runtime Environment):  JVM을 생상하며, 실행할때 필요한 라이브러리를 제공한다.
JDK(Java Development Kit): 컴파일 명령어와 실행 명령어를 담고 있다.  JRE 외에 개발에 필요한 도구들을 가지고 있다.
(명령어는 프로그램이다)
-----------------------------------------------------------------------------------------
JDK 설치
	※환경변수
	설치된 프로그램을 CLI 환경에서 명령어처럼 사용할 때에는,
	해당 경로에서만 실행할 수 있다.
	하지만 매번 해당 경로로 들어가서 실행하는 것이 불편하기 때문에
	어디서든지 실행할 수 있도록 환경 변수에 해당 경로를 설정할 수 있다.
