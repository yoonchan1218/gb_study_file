추상 클래스
	필드 안에 구현이 안된 메소드가 선언되어 있는 클래스를 추상 클래스라고 한다.
	이 때, 구현되지 않은 메소드를 추상 메소드라고 부른다.
	즉, 추상 클래스에는 추상 메소드를 선언할 수 있다.
	반드시 재정의를 통해 구현을 해야 메모리에 할당되기 때문에 "강제성"을 위해 사용한다.
	직접 인스턴스를 만들 수 없고(객체화 불가), 자식이 반드시 abstract 메서드를 구현해야 함.

추상 클래스 선언
	abstract class 클래스명 {
		abstract 리턴타입 메소드명(매개변수, ...);

		일반 메소드 선언도 가능하다.
		리턴타입 메소드명(매개변수, ...) {

		}

	}
