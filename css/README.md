# css

CSS와 클론코딩 과정에서 정리한 스타일 메모를 모아둔 폴더이다.
레이아웃, 간격, 폰트, 버튼 스타일처럼 자주 쓰는 값을 빠르게 다시 보기
위한 용도로 정리했다.

## 정리한 내용

- spacing, typography, color 관련 유틸 값 정리
- 버튼, 정렬, flex, gap, border 같은 자주 쓰는 스타일
- 클론코딩하면서 사용한 클래스 조합

## 폴더 구성

- `memo/`: 스타일 메모와 클론코딩 관련 정리
- `workspace/clonecoding/`: 직접 따라 만든 화면 작업물
- `workspace/font/`: 폰트 관련 파일
- `resource/`: 설치 파일이나 실행 환경 자료

## 확인한 메모

- `memo/kmong_clonecoding.txt`

## 공부 방식

화면을 그대로 따라 만들면서 필요한 스타일 값을 메모장에 계속 추가하는
방식으로 정리했다.
그래서 설명형 문장보다 실제로 붙여 넣을 수 있는 클래스 값이 많이 들어
있다.
