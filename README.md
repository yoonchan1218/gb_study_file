# gb_0090_kyc 공부 정리

국비 수업 들으면서 정리한 메모, 강의자료, 실습 파일을 과목별로
모아둔 폴더이다.
메모는 수업 들을 때 적은 흐름을 최대한 유지했고, 설명도 내가 이해한
방식 그대로 남겨두었다.
그래서 문장보다는 키워드 정리, 개념 설명, 실습 순서 중심으로 적힌
부분이 많다.

## 폴더 보는 기준

이 저장소는 과목마다 비슷한 구조를 가지고 있다.
메모, 강의자료, 실습 코드, 실행 환경 파일이 함께 들어 있는 경우가
많고, 각각의 의미는 다음과 같다.

- `memo/`: 수업 듣고 바로 정리한 개념 메모
- `강의자료/`: 수업 예제나 참고 파일
- `workspace/`, `study/`, `templates/`, `static/`: 직접 실습한 코드나
  화면 작업물
- `resource/`, `reesource/`, `eclipse/`: 설치 파일이나 실행 환경 자료

## 업로드 기준

GitHub에는 공부 자료가 중심이 되게 올리려고 정리했다.
설치 파일, IDE 캐시, 압축 파일, 실행 파일은 공부 기록과 직접 관련이
없어서 루트 `.gitignore`에서 제외하는 기준으로 잡았다.

## 메모 정리 방식

메모 파일은 내가 수업 들으면서 바로 정리한 흐름을 유지했다.
정의부터 적고, 특징을 적고, 필요한 경우 실습 순서를 바로 붙이는
방식으로 정리했다.
그래서 발표용 문서보다는 복습용 노트에 가깝다.

## 실습 방향

실습은 기초 문법에서 끝내지 않고, 웹과 백엔드 쪽으로 계속 이어서
확장했다.
HTML, CSS, JS를 먼저 정리하고, JSP와 Spring으로 서버 흐름을 보면서
MySQL과 Git, AWS까지 연결하는 방식으로 공부했다.
