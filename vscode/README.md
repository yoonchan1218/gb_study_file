# vscode

VSCode로 웹 기초를 시작하면서 정리한 HTML 메모와 실습 환경을 모아둔
폴더이다.
웹이 어떻게 요청과 응답으로 동작하는지, HTML과 CSS, JS가 각각 무슨
역할을 하는지부터 정리했다.

## 정리한 내용

- 서버와 클라이언트 개념
- Web, Browser, Protocol 개념
- HTTP와 HTTPS 차이
- IP와 DNS 개념
- 웹 표준
- HTML 요소와 속성
- 블록 요소, 인라인 요소, 인라인-블록 요소 차이

## 폴더 구성

- `memo/`: HTML 메모 파일
- `강의자료/`: HTML 수업 자료
- `workspace/`: 실습 과정에서 생긴 작업 폴더
- `resource/`: 설치 파일이나 실행 환경 자료

## 확인한 메모

- `memo/html.txt`
- `강의자료/html_day01/html.txt`

## 공부 방식

웹을 처음 정리할 때 필요한 기본 개념부터 차근차근 적어두었다.
태그와 속성처럼 기초이지만 자주 헷갈리는 부분은 예시 코드를 같이
적으면서 익혔다.
