# mysql

MySQL 기초와 테이블 설계 연습을 정리한 폴더이다.
단순 조회문만 보는 것이 아니라 요구사항을 보고 테이블을 어떻게 나눌지
직접 생각해보는 방식으로 공부했다.

## 정리한 내용

- 예약 시스템 테이블 설계
- 의사/환자/진료시간/예약 구조 설계
- 공유 오피스 좌석 예약 구조 설계
- 스터디 카페 좌석 예약 구조 설계
- 기본 DDL 작성 연습

## 폴더 구성

- `memo/`: SQL 메모와 설계 연습
- `workspace/ddl`, `workspace/General`: SQL 실습 폴더
- `강의자료/`: 수업 자료
- `drivers/`: DB 연결용 드라이버
- `reesource/`: 설치 파일이나 실행 환경 자료

## 확인한 메모

- `memo/sql1.txt`

## 공부 방식

문제에서 요구하는 엔티티를 먼저 표처럼 적고, 그 다음 테이블로 옮기는
방식으로 연습했다.
컬럼과 FK를 바로 SQL로 적어보면서 관계를 익히려고 했다.
