# jsp

JSP와 Servlet, 그리고 MVC 흐름을 정리한 폴더이다.
정적 페이지와 동적 페이지의 차이부터 시작해서 요청과 응답, forward와
redirect, 서블릿 흐름을 정리했다.

## 정리한 내용

- JSP 개념
- 서버, 웹 서버, 웹 컨테이너, WAS 역할
- Servlet 동작 방식
- GET, POST 차이
- forward, redirect 차이
- JSP와 MVC 실습

## 폴더 구성

- `memo/`: JSP 개념 메모
- `workspace/a-intro`, `workspace/b-mvc`: 실습 프로젝트
- `workspace_practice/`: 추가 연습 폴더
- `강의자료/`: 수업 자료
- `eclipse/`, `resource/`: 설치 파일이나 실행 환경 자료

## 확인한 메모

- `memo/jsp.txt`

## 공부 방식

요청과 응답 흐름을 이해하는 데 집중해서 정리했다.
특히 `forward`와 `redirect`처럼 자주 헷갈리는 부분은 차이를 비교하면서
메모했다.
