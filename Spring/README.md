# Spring

Spring Framework와 Spring Boot를 정리하고, 의존성 주입과 MVC 구조를
실습한 폴더이다.
단순 개념 정리에서 끝내지 않고 `controller`, `dependency`,
`threetier`, `trycatch` 같은 실습 프로젝트까지 이어서 확인했다.

## 정리한 내용

- Framework, API, 라이브러리 개념
- Spring Framework 특징
- POJO, AOP, Transaction
- DI와 Bean, ApplicationContext
- Spring Boot 기본 구조
- MVC 패턴
- 계층 구조 실습

## 폴더 구성

- `memo/`: Spring 개념 메모
- `study/`: 단계별 학습 정리 문서
- `workspace/controller`: 컨트롤러 실습
- `workspace/dependency`: 의존성 주입 실습
- `workspace/mysql`: DB 연동 실습
- `workspace/threetier`: 3계층 구조 실습
- `workspace/trycatch`: 프로젝트 실습
- `resource/`: 설치 파일이나 실행 환경 자료

## 확인한 메모

- `memo/spring.txt`
- `study/step_by_step_study.md`
- `study/study_spring_first_step.md`

## 공부 방식

Spring이 왜 필요한지 개념부터 먼저 정리하고, 그 다음 실제 프로젝트
구조를 보면서 손으로 따라 치는 방식으로 공부했다.
특히 DI와 MVC는 그림 흐름과 코드 구조를 같이 보려고 했다.
