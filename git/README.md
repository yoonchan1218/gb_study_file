# git

Git과 원격 저장소 사용 방법을 정리한 폴더이다.
로컬 작업, 브랜치 이동, merge, 원격 저장소 연결까지 순서대로 복습할 수
있게 정리했다.

## 정리한 내용

- 버전 관리 시스템 개념
- `git init`, `add`, `commit`, `log` 기본 흐름
- 브랜치 생성과 이동
- merge와 conflict 처리
- 원격 저장소 연결과 `push`, `pull`
- 협업 시작할 때 clone과 organization 흐름

## 폴더 구성

- `memo/`: Git 개념과 명령어 메모
- `workspace/career-bridge/`: Git 실습용 프로젝트
- `resource/`: 설치 파일이나 실행 환경 자료

## 확인한 메모

- `memo/git.txt`

## 공부 방식

명령어만 외우는 방식이 아니라 로컬 저장소와 원격 저장소의 흐름을 같이
보는 방식으로 정리했다.
실습 프로젝트를 하나 두고 직접 브랜치와 push, pull을 해보면서 확인했다.
