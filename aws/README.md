# aws

AWS 환경에서 Spring 프로젝트를 올리는 과정과 서버 기본 설정을 정리한
폴더이다.
로컬에서 만들던 프로젝트를 리눅스 서버 환경으로 옮길 때 필요한 순서를
메모 형태로 남겼다.

## 정리한 내용

- 타임존 설정
- OpenJDK 설치와 환경 변수 설정
- swap 설정
- Gradle 빌드와 jar 실행
- iptables를 이용한 포트 리다이렉트
- AWS에서 MySQL 연결할 때 확인할 내용

## 폴더 구성

- `강의자료/`: AWS 배포 관련 메모 파일
- `memo/`: 추가 메모 정리용 폴더
- `resource/`: 설치 파일이나 실행 환경 자료
- `workspace/`: 실습 과정에서 사용한 작업 폴더

## 확인한 메모

- `강의자료/aws_basic_hds/aws-spring.txt`
- `강의자료/aws_basic_hds/mysql_aws.txt`

## 공부 방식

명령어를 하나씩 바로 실행해보는 방식으로 정리했다.
설치, 설정, 실행 순서를 헷갈리지 않게 하려고 실제로 입력한 명령어를
그대로 적어두었다.
