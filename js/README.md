# js

자바스크립트 기본 문법과 브라우저 동작을 정리한 폴더이다.
문법만 보는 데서 끝내지 않고 DOM, 이벤트, 비동기 처리까지 단계별로
실습했다.

## 정리한 내용

- 자바스크립트 역할과 실행 방식
- 데이터 타입, 변수, 스코프
- 연산자와 함수
- 객체와 배열
- 모듈
- 비동기 처리
- 날짜 객체
- DOM과 이벤트 처리

## 폴더 구성

- `memo/`: 자바스크립트 개념 메모
- `workspace/a-intro` ~ `workspace/l-module`: 단원별 실습
- `workspace/practice`, `workspace/task02`: 추가 과제와 연습
- `강의자료/`: 수업 자료
- `resource/`: 설치 파일이나 실행 환경 자료

## 확인한 메모

- `memo/js.txt`

## 공부 방식

개념을 먼저 메모로 정리한 뒤 바로 폴더를 나눠서 실습했다.
이벤트나 DOM처럼 화면에서 바로 확인할 수 있는 내용은 직접 실행해보는
방식으로 익혔다.
